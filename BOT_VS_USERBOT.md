# 📊 Сравнение режимов: Bot vs UserBot

## Быстрая справка по возможностям

---

## 🤖 BOT РЕЖИМ (Рекомендуется)

### ✅ Что работает:

| Функция | Поддержка | Примечания |
|---------|-----------|------------|
| Сохранение удаленных сообщений в группах | ✅ Да | Бот должен быть админом |
| Сохранение фото/видео/документов | ✅ Да | В группах где админ |
| Редактирование сообщений | ✅ Да | Отслеживание изменений |
| Архивация диалогов | ✅ Да | ZIP с HTML отчетом |
| AI помощник | ✅ Да | Все провайдеры |
| Управление подписками | ✅ Да | Полная поддержка |
| Inline команды | ✅ Да | Красивый UI |
| Работа 24/7 | ✅ Да | Без ограничений |
| Безопасность | ✅ Высокая | Не нарушает ToS |

### ❌ Что НЕ работает:

| Функция | Поддержка | Причина |
|---------|-----------|---------|
| Личные сообщения других | ❌ Нет | Нет доступа к чужим ЛС |
| Самоуничтожающиеся медиа | ❌ Нет | API ограничение |
| Secret chats | ❌ Нет | E2E шифрование |
| Удаления без бота в группе | ❌ Нет | Бот должен быть членом |
| Просмотр без добавления | ❌ Нет | Требуется участие |

### ⚖️ Юридический статус

- ✅ **Легально** - не нарушает ToS Telegram
- ✅ **Безопасно** - нет риска блокировки
- ✅ **Стабильно** - официальный API

## 👤 USERBOT РЕЖИМ (⚠️ Опасно!)

### ✅ Что работает:

| Функция | Поддержка | Примечания |
|---------|-----------|------------|
| Все функции Bot режима | ✅ Да | Плюс расширенные |
| Личные сообщения | ✅ Да | **Все** ваши чаты |
| Самоуничтожающиеся медиа | ✅ Да | Фото/видео с таймером |
| Редактирование в ЛС | ✅ Да | Полная история |
| Удаленные из любых чатов | ✅ Да | Где вы участник |
| Массовые удаления | ✅ Да | Автоархивация |
| Работа без админских прав | ✅ Да | Как обычный пользователь |

### ❌ Что НЕ работает:

| Функция | Поддержка | Причина |
|---------|-----------|---------|
| Secret chats | ⚠️ Частично | Ограничения MTProto |
| Чаты других пользователей | ❌ Нет | Только ваш аккаунт |
| Сообщения до запуска | ❌ Нет | Только новые события |

### ⚖️ Юридический статус

- ❌ **НАРУШАЕТ ToS Telegram**
- ❌ **Риск бана** - может быть заблокирован аккаунт
- ⚠️ **Нестабильно** - неофициальный API

### 🚨 РИСКИ:

1. **Перманентная блокировка** аккаунта
2. **Потеря доступа** ко всем чатам
3. **Юридические последствия** при неправомерном использовании
4. **Нарушение приватности** других пользователей

## 💡 Практические сценарии

### Сценарий 1: Работа с командой

**Задача:** Сохранять важные сообщения в рабочем чате

**Решение:** Bot режим
- Добавьте бота в групповой чат как админа
- Настройте сохранение
- Все легально и безопасно

### Сценарий 2: Личная переписка

**Задача:** Сохранять личные сообщения

**Решение 1 (безопасно):**
- Создайте группу с собой и нужным человеком
- Добавьте бота как админа
- Попросите друга согласие

**Решение 2 (рискованно):**
- UserBot с **отдельным** аккаунтом
- Только для критически важных случаев
- Готовность к блокировке

### Сценарий 3: Архивация больших групп

**Задача:** Сохранить историю большого чата перед его удалением

**Решение:** Гибридный режим
- Bot для групповых сообщений
- UserBot для личных (если критично)
- Максимальный охват
