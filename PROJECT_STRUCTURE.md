## 📦 Минимальный набор для запуска

Всего **3 файла** для старта:

1. `merai_bot.py` - основной код
2. `requirements.txt` - зависимости
3. `.env` - конфигурация (создать из .env.example)

```bash
# Быстрый старт
cp .env.example .env
nano .env  # Добавьте BOT_TOKEN
pip install -r requirements.txt
python merai_bot.py
```

## 🔄 Обновления

### Как обновить бота?

```bash
# Сохраните свою конфигурацию
cp .env .env.backup

# Обновите код
git pull origin main

# Восстановите конфигурацию
cp .env.backup .env

# Переустановите зависимости (если изменились)
pip install -r requirements.txt --upgrade

# Перезапустите бота
# (зависит от метода развертывания)
```

## 🛠 Кастомизация

### Изменение планов подписки

Отредактируйте в `merai_bot.py`:
```python
SUBSCRIPTION_PLANS = {
    'starter': {
        'price_stars': 100,  # Измените цену
        'days': 7,           # Измените период
        ...
    }
}
```

### Добавление своего AI провайдера

Добавьте метод в класс `AIAssistant`:
```python
@staticmethod
async def _your_ai_request(api_key, prompt, message):
    # Ваша реализация
    pass
```

### Изменение текстов

Все тексты находятся в `merai_bot.py`:
- Приветственное сообщение
- Справка
- Уведомления
- И т.д.

## 🔐 Безопасность

### Что защищено:
- ✅ API ключи в .env (не в Git)
- ✅ Токены ботов (environment variables)
- ✅ Session файлы (в .gitignore)
- ✅ База данных (локальное хранение)

### Что НЕ защищено:
- ❌ Содержимое сообщений (не шифруется)
- ❌ Метаданные (видны в логах)

### Рекомендации:
1. Используйте сильные пароли
2. Не делитесь .env файлом
3. Регулярно обновляйте зависимости
4. Включите 2FA в Telegram
