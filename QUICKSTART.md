# 🚀 БЫСТРЫЙ СТАРТ - MerAi & Monitoring

## За 5 минут до запуска

### ШАГ 1: Скачать и установить

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/merai-monitoring.git
cd merai-monitoring

# Автоматическая установка (Linux/Mac)
chmod +x setup.sh
./setup.sh

# ИЛИ вручную (Windows)
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### ШАГ 2: Получить токен бота

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен (выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### ШАГ 3: Настроить конфигурацию

```bash
# Скопировать пример конфигурации
cp .env.example .env

# Отредактировать (выберите редактор)
nano .env        # Linux/Mac
notepad .env     # Windows
```

**Минимальная конфигурация:**
```env
BOT_TOKEN=ваш_токен_от_botfather
```

**Для UserBot режима (ОПАСНО!):**
```env
BOT_TOKEN=ваш_токен_от_botfather
API_ID=ваш_api_id
API_HASH=ваш_api_hash
```

> Получить API_ID и API_HASH: https://my.telegram.org/apps

### ШАГ 4: Запустить!

```bash
# Активировать окружение (если еще не активировано)
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Запустить бота
python merai_bot.py
```

### ШАГ 5: Выбрать режим

При запуске выберите:
1. **Bot режим** (РЕКОМЕНДУЕТСЯ) - безопасно
2. **UserBot режим** (⚠️ нарушает ToS Telegram)
3. **Гибридный режим** - оба одновременно

### Bot режим

- ❌ Может привести к **ПЕРМАНЕНТНОМУ БАНУ**
- ❌ Telegram **МОЖЕТ ЗАБЛОКИРОВАТЬ** ваш аккаунт
- ❌ Администрация **НЕ НЕСЕТ ОТВЕТСТВЕННОСТИ**

**Используйте ТОЛЬКО:**
- Отдельный аккаунт (не основной)

- Сохраняя чужие сообщения, вы несете ответственность

## 🐛 Что делать если не работает?

### Бот не запускается

```bash
# Проверить токен
nano .env

# Проверить зависимости
pip install -r requirements.txt

# Проверить Python версию (нужна 3.12+)
python --version
```

### Не перехватывает удаления

**Bot режим:**
- Проверьте, что бот **АДМИН** в группе
- Дайте права на удаление сообщений

**UserBot режим:**
- Проверьте правильность API_ID и API_HASH
- Убедитесь, что авторизация прошла

### Ошибка при оплате

- Проверьте, что у вас достаточно Stars
- Убедитесь, что платежи не заблокированы в Telegram
