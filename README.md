## ⚙️ Конфигурация

### Получение Bot Token

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен в `.env` файл:
   ```
   BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ
   ```

### Получение API_ID и API_HASH (для UserBot)

⚠️ **ТОЛЬКО ЕСЛИ ВЫ ПОНИМАЕТЕ РИСКИ!**

1. Перейдите на [my.telegram.org](https://my.telegram.org)
2. Войдите с вашим номером телефона
3. Перейдите в "API development tools"
4. Создайте новое приложение
5. Скопируйте `api_id` и `api_hash`:
   ```
   API_ID=12345678
   API_HASH=abcdef1234567890abcdef1234567890
   ```

### Настройка AI ключей

#### Gemini (Google)

1. Перейдите на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Создайте API ключ
3. Добавьте в `.env`:
   ```
   GEMINI_API_KEY=ваш_ключ_здесь
   ```

#### Другие AI

- **Grok:** Получите доступ на сайте xAI
- **GLM:** ChatGLM API

## 🎭 Режимы работы

### 🤖 Bot режим (РЕКОМЕНДУЕТСЯ)

**Плюсы:**
- ✅ Безопасно - не нарушает ToS
- ✅ Стабильно
- ✅ Официальная поддержка

**Минусы:**
- ❌ Не видит личные сообщения других пользователей
- ❌ Не может перехватывать самоуничтожающиеся медиа
- ❌ Требует добавления в группы как админа

**Что работает:**
- Сохранение удалений в группах/каналах (где бот админ)
- AI ассистент
- Управление подписками

### 👤 UserBot режим (ОПАСНО!)

⚠️ **ВНИМАНИЕ:** Нарушает ToS Telegram!

**Плюсы:**
- ✅ Полный доступ ко всем сообщениям
- ✅ Перехват самоуничтожающихся медиа
- ✅ Работа в личных чатах

**Минусы:**
- ❌ НАРУШАЕТ ToS Telegram
- ❌ РИСК ПЕРМАНЕНТНОГО БАНА
- ❌ Нестабильно
- ❌ Может привести к юридическим проблемам

**Что работает:**
- Все функции Bot режима +
- Перехват в личных чатах
- Самоуничтожающиеся сообщения
- Secret chats (с ограничениями)

### 🔄 Гибридный режим

Запуск обоих режимов одновременно для максимальной функциональности.

## ❓ FAQ

### Q: Бот не запускается?

**A:** Проверьте:
1. Правильность токена в `.env`
2. Установлены ли все зависимости: `pip install -r requirements.txt`
3. Версию Python: `python --version` (должна быть 3.12+)

### Q: Не работает перехват удалений?

**A:** 
- **Bot режим:** Бот должен быть админом в группе
- **UserBot режим:** Проверьте правильность API_ID и API_HASH
- Убедитесь, что авторизация прошла успешно

### Q: Аккаунт заблокирован Telegram?

**A:** К сожалению, это риск UserBot режима. Мы предупреждали! Используйте только Bot режим или отдельный аккаунт для UserBot.

### Q: Как добавить свой AI?

**A:** Зайдите в админ-панель (`/admin`) → AI Ключи. Или отредактируйте `.env` файл.

### Q: Можно ли вернуть деньги за подписку?

**A:** Нет, все платежи окончательные согласно [Условиям использования](TERMS_AND_PRIVACY.md).

### Q: Как работает функция связанных ботов?

**A:** Premium пользователи могут подключить свой бот-токен, что даст +7 дней к подписке. Ваш бот будет дублировать функции основного.

## 🛠 Разработка

### Структура проекта

```
merai-monitoring/
├── merai_bot.py          # Основной файл (всё в одном)
├── requirements.txt      # Зависимости
├── .env.example          # Пример конфигурации
```

### Минималистичная архитектура

Весь код в одном файле для упрощения развертывания:
- `Config` - конфигурация
- `Database` - JSON база данных
- `AIAssistant` - AI интеграция
- `BotMode` - Bot API (aiogram)
- `UserBotMode` - MTProto (Pyrogram)
