## 🚀 Как начать? (3 простых шага)

### ШАГ 1: Получи токен бота

1. Открой Telegram
2. Найди [@BotFather](https://t.me/BotFather)
3. Отправь `/newbot`
4. Следуй инструкциям
5. Скопируй токен (выглядит так: `1234567890:ABC...`)

### ШАГ 2: Настрой конфигурацию

```bash
# Скопируй пример конфигурации
cp .env.example .env

# Отредактируй файл
nano .env  # Linux/Mac
notepad .env  # Windows
```

**Минимальная настройка (.env):**
```env
BOT_TOKEN=твой_токен_от_botfather
```

### ШАГ 3: Запусти!

**Вариант А - Автоматическая установка (Linux/Mac):**
```bash
chmod +x setup.sh
./setup.sh
# Следуй инструкциям
python merai_bot.py
```

**Вариант Б - Вручную (Windows/Linux/Mac):**
```bash
# Создай виртуальное окружение
python -m venv venv

# Активируй
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Установи зависимости
pip install -r requirements.txt

# Запусти
python merai_bot.py
```

## 🔧 Настройка дополнительных функций

### AI интеграция

Чтобы включить AI помощника, добавь в `.env`:

```env
# Gemini (Google) - получи ключ на https://makersuite.google.com/app/apikey
GEMINI_API_KEY=твой_ключ

# Или настрой через админ-панель в боте
```

### UserBot режим (если ДЕЙСТВИТЕЛЬНО нужен)

⚠️ **ВНИМАНИЕ:** Используй ТОЛЬКО отдельный аккаунт!

1. Получи API_ID и API_HASH на https://my.telegram.org/apps
2. Добавь в `.env`:
   ```env
   API_ID=12345678
   API_HASH=abcdef1234567890
   ```
3. При запуске выбери режим 2 или 3
4. Авторизуйся с номером телефона

## 🐛 Частые проблемы

### Бот не запускается?
1. Проверь, что токен правильный в `.env`
2. Убедись, что установлены зависимости: `pip install -r requirements.txt`
3. Проверь версию Python: `python --version` (нужна 3.12+)

### Не перехватывает удаления?
**Bot режим:**
- Добавь бота в группу как **администратора**
- Дай права на удаление сообщений

**UserBot режим:**
- Проверь правильность API_ID и API_HASH
- Убедись, что авторизация прошла успешно

### Ошибка "Module not found"?
```bash
pip install -r requirements.txt --upgrade
```
