## 2. ОПИСАНИЕ СЕРВИСА

2.1. MerAi & Monitoring - это программное обеспечение для работы с Telegram, предоставляющее следующие функции:
- Сохранение удаленных сообщений
- Архивация диалогов
- AI-ассистент для анализа сообщений
- Управление Telegram каналами и группами

2.2. Сервис работает в двух режимах:
- **Bot режим** - использует официальный Telegram Bot API
- **UserBot режим** - использует Telegram MTProto API (пользовательский клиент)

## 5. ПЛАТЕЖИ И ПОДПИСКИ

### 5.1. Способы оплаты

Сервис поддерживает:
- Telegram Stars (встроенная валюта Telegram)
- Банковские карты (через Telegram Payments)
- Криптовалюты (опционально, настраивается администратором)

### 5.2. Возвраты

- Все платежи окончательные и НЕ ПОДЛЕЖАТ ВОЗВРАТУ
- Исключение: технические ошибки при обработке платежа
- Возврат осуществляется в течение 14 дней на усмотрение администрации

### 5.3. Подписки

- Подписка начинается сразу после оплаты
- Автопродление доступно только для некоторых планов
- Вы можете отменить автопродление в любой момент
- При отмене подписка действует до окончания оплаченного периода
